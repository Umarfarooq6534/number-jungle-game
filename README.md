# 🌿 Number Jungle 🐒

**Number Jungle** is a fun and interactive math game designed for kids to practice addition and subtraction. Help Momo the Monkey count fruits and animals correctly!  

This project was built using **HTML, CSS, and JavaScript** and is fully playable in the browser, including on mobile devices.

---

## 🎮 Features

- Add, subtract, or mix math questions
- Interactive visual counters (fruits, animals)
- Fun animations and soundless feedback
- Score, streaks, and level progression
- Works on PC and mobile

---

## 📂 Technologies Used

- **HTML** – Structure and layout  
- **CSS** – Styling, animations, and responsive design  
- **JavaScript** – Game logic, scoring, and interactivity
